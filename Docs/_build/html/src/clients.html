static AltaClientes()[fuente]¶

Modulo que carga los datos del cliente

Devuelve

None

Tipo del valor devuelto

None

Se crea una lista newcli que contendrá todos los datos del cliente que se introduzcan en lo widgets, esta lista se pasa cmo argumento al módulo altaCli del módulo conexión. Este módulo llama a la función crear clientes y recarga la tabla con todos los clientes, y finalmente, el módulo llama a la función limpiarCli, que vacía el contenido de los widgets.

Devuelve

Tipo del valor devuelto

Pone la letra inicial en mayúsculas, comprueba que son nueve caracteres. Toma los 8 primeros, si es extranjero cambia la letra por el número, y aplica el algoritmo de comprobación de la letra basado en la normativa Si es correcto devuelve true, si es falso devuelve false
