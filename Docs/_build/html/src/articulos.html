articulos module¶

class articulos.Articulos[fuente]¶

Bases: object

static AltaArticulos()[fuente]¶

Módulo que carga los datos de los artículos

Devuelve

None

Tipo del valor devuelto

None

Guarda los datos en art, añade una nueva fila con .insertRow(), guarda los datos en la base de datos con el metodo .altaArt(newart) y recarga la tabla de los articulos para que aparezcan los nuevos productos.

static bajaArticulo()[fuente]¶

Módulo con el que elegimos el producto que va a ser borrado según el código que seleccionemos.

Devuelve

None

Tipo del valor devuelto

None

Tras seleccionar el producto en la tabla, aprovechamos que están cargados los datos y lo borramos cogiendo el código del producto. Llamamos al metodo confirmBorrarProducto para que el usuario confirme el borrado del producto, y si es así, recargamos la tabla de los articulos y limpiamos la seccion de productos.

static cargarArt()[fuente]¶

Módulo que carga los datos del articulo seleccionado, para poder hacerle modificaciones o, ver su precio, codigo, etc

Devuelve

None

Tipo del valor devuelto

None

Guardamos los datos del artículo seleccionado y llamamos al metodo de conexión para que nos muestre los datos del articulo

static limpiarProd()[fuente]¶

Metodo que limpia la sección de artículos.

Devuelve

None

Tipo del valor devuelto

None

Guardamos todas la variables que vamos a limpiar en articulos[] y posteriormente, lo recorremos con un bucle «for» y ponemos los editText y labels a vacío.

static modifArticulos()[fuente]¶

Método para modificar algun dato de los artíctulos (puede ser cualquier dato, menos el código del producto)

Devuelve

None

Tipo del valor devuelto

None

Cargamos todos los editText (ya que, se pudo haber modificado más de uno) y llamamos a confirmModificación para que el usuario confirme los cambios y poder así, actualizar los datos en la BBDD. Luego recargamos la tabla de articulos para poder ver los datos que hemos modificado.

static reloadArt()[fuente]¶

Módulo que limpia y recarga la tabla de artículos.

Devuelve

None

Tipo del valor devuelto

None

Llamamos al método limpiarProd() para dejar limpia la sección de artículos y posteriormente, recargamos la tabla de artículos con el método de .mostrarArticulos()
